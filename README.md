# Návrh a implementácia cross-platformovej mobilnej aplikácie pre kontrolu prístupu
Práce se zabývá návrhem a implementací cross-platformové mobilní aplikace pro kontroly přístupu do budov. Výsledný prototyp umožní centralizovanou správu přístupových práv v hierarchické struktuře organizace, digitální odemykání zámků a přiřazení oprávnění uživatelům. Práce analyzuje požadavky na moderní systémy kontroly přístupu, navrhuje architekturu řešení a implementuje funkční prototyp s důrazem na bezpečnost a intuitivnost používání. Výsledkem bude mobilní aplikace kompatibilní s iOS a Android platformami představující moderní alternativu k tradičním mechanickým klíčům a kartovým systémům.
